LaserHarp
=========

Originally written by: Stephen Hobley

Modified by: Josiah Witt

analog_digital_test - prints values of analog and digital inputs side by side to test if theremin is wired correctly.

midi_test - sends MIDI notes to test that MIDI is setup correctly.

theremin - single string harp. Works by itself. Intended as a test to ensure you have it working.

control_harp - basic harp control program.

vel_harp - modified harp control program whichs maps hand position to MIDI velocity.
